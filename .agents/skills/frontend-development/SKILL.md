# Frontend Development Skill

## Skill Purpose
Develop a high-performance horror-themed website frontend that combines cinematic visuals, smooth animations, responsiveness, and modern architecture.

---

# Core Technologies

- React.js
- Tailwind CSS
- Framer Motion
- Vite / Next.js
- React Router
- Axios

---

# Primary Responsibilities

## 1. Component-Based Architecture
Develop reusable and scalable UI components.

### Components Include
- Hero section
- Download cards
- Screenshot galleries
- Feedback forms
- Navigation bars
- Footer sections

---

## 2. Responsive Development
Ensure the website adapts seamlessly across all devices.

### Requirements
- Mobile-first design
- Adaptive layouts
- Flexible grids
- Responsive typography

---

## 3. Performance Optimization
Maintain smooth performance despite heavy visual effects.

### Optimization Areas
- Lazy loading
- Code splitting
- Asset compression
- Optimized rendering
- GPU-accelerated animations

---

## 4. Accessibility
Ensure the website remains usable and readable.

### Accessibility Goals
- Semantic HTML
- Keyboard navigation
- Proper contrast ratios
- Screen reader compatibility

---

# Best Practices

- Modular folder structure
- Reusable hooks and utilities
- Clean naming conventions
- Consistent coding standards

---

# Expected Output

- Fast and responsive frontend
- Scalable architecture
- Immersive horror-themed experience

# Feedback & Bug Reporting Skill

## Objective
Build a player feedback and bug reporting system for community interaction.

## Responsibilities
- Create feedback forms
- Develop bug reporting workflows
- Enable feature requests
- Store reports securely
- Support admin review systems

## Form Fields
- Name
- Email
- Bug Title
- Description
- Severity
- Screenshot Upload
- Suggestions

## Backend Compatibility
- FastAPI
- Node.js
- MongoDB
- PostgreSQL

## UX Goals
- Easy reporting
- Clear form structure
- Fast submission flow

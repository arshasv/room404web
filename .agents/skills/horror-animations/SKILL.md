# Horror Animation & Visual Effects Skill

## Skill Purpose
Develop cinematic horror-themed animations and atmospheric visual effects that increase suspense and immersion.

---

# Core Objectives

- Enhance emotional tension
- Create cinematic atmosphere
- Improve visual storytelling
- Maintain smooth performance

---

# Primary Responsibilities

## 1. Atmospheric Effects
Add subtle environmental animations.

### Effects Include
- Smoke overlays
- Fog movement
- Ambient particles
- Shadow motion
- Floating dust

---

## 2. Horror Interactions
Create interactive horror effects.

### Examples
- Flickering lights
- Glitch transitions
- Blood-drip animations
- Distortion effects
- Hover-based suspense effects

---

## 3. Transition Design
Create cinematic transitions between sections.

### Goals
- Smooth scrolling
- Dramatic section reveals
- Tension-building movement

---

# Tools

- Framer Motion
- CSS Animations
- GSAP

---

# Expected Output

- Suspenseful cinematic motion system
- Smooth high-performance animations
- AAA horror game presentation quality

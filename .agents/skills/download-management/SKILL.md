# Download Management Skill

## Objective
Provide seamless game downloads for Windows and Ubuntu platforms.

## Responsibilities
- Create platform-based downloads
- Display version information
- Show system requirements
- Manage downloadable assets
- Handle secure file delivery

## Supported Platforms
- Windows
- Ubuntu/Linux

## Features
- Download cards
- Platform icons
- Version display
- File size display
- Download analytics

## Security Goals
- Secure hosting
- Verified files
- Protected download endpoints

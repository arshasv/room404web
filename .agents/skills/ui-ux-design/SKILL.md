# Horror-Themed UI/UX Design Skill

## Skill Purpose
This skill focuses on designing an immersive horror survival game website experience that emotionally connects users to the atmosphere of the game. The design must create fear, suspense, mystery, and curiosity while remaining visually modern and user-friendly.

---

# Core Objectives

- Create a cinematic horror experience
- Maintain visual consistency with the game
- Build emotional engagement through design
- Increase user immersion and suspense
- Guide users toward downloads and community interaction

---

# Primary Responsibilities

## 1. Horror Atmosphere Creation
Design interfaces that feel dark, unsettling, mysterious, and cinematic.

### Required Elements
- Dark color palettes
- Blood-red highlights
- Cinematic shadows
- Distressed textures
- Fog/smoke overlays
- Ambient lighting effects

---

## 2. User Experience Design
Create layouts that are easy to navigate while maintaining immersion.

### UX Goals
- Clear visual hierarchy
- Smooth navigation
- Strong call-to-action visibility
- Minimal distraction
- Emotional storytelling through UI

---

## 3. Typography & Branding
Typography should visually match horror survival games.

### Typography Style
- Distressed fonts for titles
- Sharp cinematic headings
- Readable body text
- Glowing red hover effects

---

## 4. Responsive Layout Design
Ensure immersive experiences across all screen sizes.

### Supported Devices
- Desktop
- Laptop
- Tablet
- Mobile

---

# Recommended Tools

- Figma
- Adobe XD
- Photoshop
- Professional responsive UI system

# Design Rules

## Rule 1 — Maintain Horror Atmosphere
Every visual element on the website must contribute to the horror survival theme.

### Explanation
The website should emotionally feel like the game itself. Users should immediately experience tension, suspense, darkness, and mystery.

### Allowed
- Dark environments
- Red glowing accents
- Distressed textures
- Cinematic shadows
- Atmospheric fog

### Avoid
- Bright cheerful colors
- Cartoon-like designs
- Overly playful visuals
- Generic corporate UI

---

## Rule 2 — Cinematic Consistency
Maintain visual consistency throughout the entire website.

### Explanation
All pages must feel connected to the same horror universe.

### Requirements
- Consistent color palette
- Consistent typography
- Consistent animation style
- Consistent spacing and layout systems

---

## Rule 3 — Emotional Storytelling
The UI should communicate fear, mystery, and survival.

### Implementation
- Use suspense-driven layouts
- Reveal content dramatically
- Use ambient imagery
- Keep strong visual focus

---

## Rule 4 — Minimal Yet Immersive Layouts
Avoid cluttered interfaces.

### Explanation
Too many elements reduce suspense and cinematic quality.

### Goals
- Strong focus areas
- Clean spacing
- Limited distractions
- High readability

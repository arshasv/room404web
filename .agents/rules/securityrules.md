# Security Rules

## Rule 1 — Input Sanitization
Sanitize all form inputs before processing.

---

## Rule 2 — Prevent Spam & Abuse
Protect forms using CAPTCHA and rate limiting.

---

## Rule 3 — Secure Downloads
Downloadable game files must be verified and protected.

---

## Rule 4 — Secure Database Storage
Sensitive information must not be exposed publicly.

---

## Rule 5 — API Protection
Protect APIs against abuse and unauthorized access.
